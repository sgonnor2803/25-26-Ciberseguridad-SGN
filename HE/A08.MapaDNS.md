## 1. ***Introducción***

En este proyecto se llevará a cabo un ***mapa DNS y un análisis OSINT*** de la empresa "El POZO". La idea es **realizar una investigación inicial usando únicamente fuentes públicas**, para recopilar información sobre sus dominios, subdominios, servidores DNS y de correo, así como otros datos relevantes que nos permitan entender la infraestructura digital de la empresa.

Este proceso servirá como **fase preparatoria de una auditoría ética**, permitiendo tener una visión clara de la empresa desde el exterior antes de pasar a pruebas más específicas.

---
## 2. ***Objetivo del Proyecto***

El objetivo principal es **recopilar, organizar y documentar toda la información relevante** sobre la infraestructura digital de “El POZO”. Esto incluye:

- Dominios y subdominios asociados al dominio principal.
- Servidores DNS y de correo electrónico.
- Información adicional que pueda ser útil para futuras auditorías, como presencia en redes sociales o datos de contacto públicos.

Con esta información se pretende **identificar posibles puntos de interés o vulnerabilidades externas**, siempre respetando la legalidad y usando únicamente técnicas OSINT.

## 4. ***Recopilación de Información***

---
### 4.1 ***Información Whois***
---

Se recopiló información pública del dominio principal de la empresa, `elpozo.com`, usando consultas Whois y resolviendo su IP.  
El objetivo era entender quién gestiona el dominio, dónde está alojado y qué proveedor usa, todo desde fuentes abiertas.

#### ***1) Consulta del dominio***

Primero se miraron los datos del dominio con:

```bash
whois elpozo.com
```

<img width="1122" height="576" alt="image" src="https://github.com/user-attachments/assets/621c6185-3b89-40f7-9f59-0a01b1eab9ea" />

Con esto se obtuvieron datos como el registrador, las fechas de creación y expiración, el estado del dominio y los servidores de nombres.

#### ***2) Obtención de la dirección IP***

Para conocer la IP del dominio se usó:

```bash
dig elpozo.com
```

<img width="1118" height="512" alt="image" src="https://github.com/user-attachments/assets/6263c140-1510-4c23-b573-cc13bf52ac33" />

La IP que se obtuvo fue `217.76.132.167`.

#### ***3) Consulta Whois de la IP***

Con la IP se hizo otra consulta Whois para conocer el sistema autónomo, el rango de red y el proveedor:

```bash
whois 217.76.132.167
```

<img width="1119" height="1007" alt="image" src="https://github.com/user-attachments/assets/93c3c0b0-8c40-44d0-b729-2b0f966f5c3b" />

---

#### ***4) Resumen de la información obtenida***

| Campo | Información |
|--------|------------|
| Dominio | elpozo.com |
| Dirección IP principal | 217.76.132.167 |
| Rango de red | 217.76.132.128 - 217.76.132.255 |
| Sistema Autónomo (AS) | AS8560 |
| ISP | Arsys Internet, S.L. (arsys.es) |
| País | España |
| Registrador | Arsys Internet, S.L. (NICLINE.COM) |
| Servidores de nombres | scen15.elpozo.com, scen17.elpozo.com |
| Correo de abuso | abuse@arsys.es |
| DNSSEC | No firmado |

Esta información permite identificar quién controla el dominio y cómo está organizada la infraestructura desde fuera. Conocer la IP, el AS y el proveedor ayuda a entender la red y sirve como base para cualquier análisis posterior. Es toda información pública, pero muy útil para tener un panorama claro.
